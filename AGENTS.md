# Agent guidelines

## Copy style

Use **sentence case** for all user-facing copy — headings, labels, buttons, nav items, page titles.
Only the first word is capitalized. Proper nouns and product/package names keep their own casing
(e.g. `AiiDA`, `AiiDAlab`, `Materials Cloud`, `aiida-core`, `GitHub`, `REST API`, `FAIR`).

- Yes: `Graph gallery`, `Latest news`, `Science & publications`, `Scalable & performant`
- No: `Graph Gallery`, `Latest News`, `Science & Publications`, `Scalable & Performant`
